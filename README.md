## TXCS
ToastXC Services/Software
Free, open source, performant, Linux first.

These are small command line programs, created for RHEL based servers and used by toastxc.xyz and related services.
### Install
```bash
sh install-all.sh
```
### Update
```bash
sh update-all.sh
```
### Remove
```bash
sh remove-all.sh
```
